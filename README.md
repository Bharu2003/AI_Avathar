# Lightweight AI Coach Orchestrator Website

A polished Streamlit frontend for the Lightweight AI Coach Orchestrator concept.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## What this UI demonstrates

- Session setup (age group, role, tone, language, goal)
- Rule-based coach assignment preview (Coach Tara / Coach Ravi)
- Simple live chat interaction mockup
- Architecture + endpoint overview for FastAPI integration
- Clean visual layout suitable for demo and stakeholder walkthroughs
